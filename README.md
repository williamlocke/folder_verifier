Folder Verifier
===============


Installation
------------

~~~ sh
$ git clone git@github.com:williamlocke/folder_verifier.git
$ cd folder_verifier
$ rake install
~~~



Usage
-----

Verify a given directory
~~~ sh
$ folder_verifier verify Images
~~~

Print a directory structure
~~~ sh
$ folder_verifier print Images
~~~


Help
----

View available commands
~~~ sh
$ folder_verifier 
~~~
